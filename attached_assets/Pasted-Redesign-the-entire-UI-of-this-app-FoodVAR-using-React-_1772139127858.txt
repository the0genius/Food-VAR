MOTION & ANIMATIONS (Framer Motion Required)

- Use Framer Motion for all primary interactions. 
- Cards and Buttons on press: scale down slightly (scale: 0.97) using spring physics.
- Page transitions and UI reveals: slide up and fade in with spring easing (stiffness: 260, damping: 20).
- Score gauge: animates from 0 to final value smoothly on mount, with a dynamic color transition.
- Search results: staggered fade-in list (delayChildren: 0.05).
